========================================================================================
		         	 回归模板使用说明
========================================================================================

ipynb文件（共五个）
1. stratified_backtest.ipynb
回测模板，回测模板的需要修改文件保存的路径，跑出的回归结果会保存在指定路径下的test.csv文件中
此模板会自动保存回测因子的相关图像到文件路径下
2. 因子、基金、股指月度收益率数据以及基金和股指的回归.ipynb
此模板读取test.csv下的日度数据，同时自动从聚宽因子库中下载风险因子每日数据，保存至index文件中。
自动读取路径下的fund.csv文件获取每日基金的净值数据。
最后会将上述所有因子、收益率、指数等数据统一处理成月度数据输出到month.csv
此模板最后一部分是使用股指和收益率进行的回归，目的是检验哪一个股指更适合作为基准，可以不运行。
3. 多元线性回归.ipynb
此模板对超额收益率和因子进行回归，通过此回归结果来判断因子和阿尔法的显著性
4. 时间序列检验.ipynb
此模板是对新得到的时间序列数据进行平稳性检验，若不平稳，需要进行差分处理。目前所有得到的因子已经通过平稳化检验。
5. beta系数回归.ipynb
此模板是在确定参与回归的因子之后进行的最后一步，由此可以计算出巴菲特积极策略的收益率，并将最终结果输出到month.csv中

本地csv文件（四个）
1. index.csv
存储聚宽下载的指数数据，一般不需要修改。
2. fund.csv
存储东方红四号基金的收益率，一般不需要修改。
3. test.csv 
每次运行回测之后的数据将保存在test.csv中，其中包含了各种因子的每日数据。其中的数据既可以通过回测模板输入，也可以手动复制factor garbage表格中已有的数据。
4. month.csv
从test.csv中读取因子的每日数据，在"因子、基金、股指月度收益率数据以及基金和股指的回归.ipynb"中处理为月度数据之后就会放置与month.csv中，
其中保存了因子的月度数据。
5. factor garbage
此表格不参与程序运行，其中保存了一些回归起来并不显著的因子数据，需要时可以直接把其中的因子添加到test.csv中。

注意事项
1. 如果需要从聚宽因子库中添加新的数据，需要使用stratified_backtest获得数据输入到test.csv中
2. 如果已经在factor garbage中有相关的数据，则可以手动从中添加因子至test.csv中
3. 进行多元回归时，需要修改最前面的independent列表添加想要回归的自变量。
4. 需要注意在运行“因子、基金、股指月度收益率数据以及基金和股指的回归“这一过程中需要关闭test.csv文件
5. 如果要进行多元回归，需要先把需要的数据保存在test.csv中，然后运行上面第二个ipynb，将数据处理为月度数据输送至month.ipynb
，然后运行上面第三个ipynb，得到回归结果
6. 修改后的回测模板会自动保存回测过程中的图像到路径中。
